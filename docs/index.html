⬇ 최신 버전 다운로드 5분 만에 시작하기

Windows · 무료 · 오픈소스(GPL-3.0) · 릴리즈에서 setup.exe 하나만 받으면 됩니다

🚀 5분 만에 시작하기

설치부터 첫 이미지까지, 이 순서면 끝납니다.

1

설치

릴리즈 페이지에서 setup.exe를 받아 실행하세요. 기존 NAIS3를 쓰던 분은 데이터가 그대로 이어집니다.

2

토큰 연결

NovelAI 사이트 → 계정 설정 → Get Persistent API Token 복사 → 앱 설정(⚙) → 토큰에 붙여넣기. 토큰은 내 PC에만 암호화 저장됩니다.

3

첫 생성

왼쪽에 프롬프트를 쓰고 생성 버튼! 한글로 "핑크 동공"처럼 쳐도 태그를 추천해줍니다.

4

씬으로 대량 생성

상단 씬 탭에서 프롬프트를 카드로 저장하고 예약을 쌓으면, 수십·수백 장이 자동으로 뽑힙니다.

🧭 화면 둘러보기

상단 네비게이션의 6개 페이지가 각각 뭘 하는 곳인지.

메인 기본 생성

프롬프트·캐릭터·바이브를 왼쪽에서 설정하고 한 장씩 생성하는 기본 화면. 가운데 큰 미리보기, 오른쪽 히스토리. 인페인트·i2i도 여기서.

첫 생성 방법 →

씬 대량 생성

프롬프트+해상도를 카드로 저장하고 예약을 쌓아 자동 생성. 편집 모드로 수십 개 씬을 한 번에 조작하고, 번호 매기기·폴더 내보내기까지. 이 앱의 심장입니다.

씬 사용법 → 편집 모드 →

디렉터 · 라이브러리 도구

디렉터는 이미지 후처리 도구, 라이브러리는 참고 이미지를 모아두는 갤러리입니다. 라이브러리의 이미지는 바이브·레퍼런스 재료로 쓰기 좋아요.

라이브러리 → 스타일 분석 →

웹 내장 브라우저

단보루 같은 태그 사이트를 앱 안에서. 퀵링크 등록, 로그인 유지, 프록시 우회까지. 태그 구경하다가 바로 프롬프트로.

웹 모드 → 한글 태그 검색 →

업로드 Cloudflare R2

이미지를 R2 버킷에 개수 제한 없이 바로 업로드. 폴더 탐색, 드래그 앤 드롭, 실패 재시도, 즐겨찾기 별명까지. 100개 제한은 잊으세요.

R2 업로드 → 연결 방법 →
